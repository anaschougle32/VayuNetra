API CLASSIFICATION:
===================

🔴 CRITICAL APIS (Must Enable - Core Functionality):
----------------------------------------------

1. Maps JavaScript API
   - Purpose: Interactive maps for web application
   - Usage: All map displays, trip logging, location management
   - Status: ALREADY IMPLEMENTED

2. Places API
   - Purpose: Location autocomplete and place details
   - Usage: Search for start/end locations, home/work addresses
   - Status: ALREADY IMPLEMENTED

3. Directions API
   - Purpose: Route calculation and directions
   - Usage: Calculate trip routes, distances, travel times
   - Status: ALREADY IMPLEMENTED

4. Geocoding API
   - Purpose: Convert addresses to coordinates and vice versa
   - Usage: Address validation, coordinate conversion
   - Status: ALREADY IMPLEMENTED

5. Air Quality API
   - Purpose: Air quality data for specific locations
   - Usage: Pollution dashboard, environmental impact calculations
   - Resolution: 500x500 meters
   - Status: NEEDED FOR POLLUTION FEATURES

6. Pollen API
   - Purpose: Pollen data for specific locations
   - Usage: Environmental health factors, trip quality scoring
   - Resolution: 1x1km
   - Status: NEEDED FOR ENVIRONMENTAL FEATURES

   - Purpose: Static map images

   - Status: NICE TO HAVE

   - Status: ENHANCEMENT FEATURE

9. Time Zone API
