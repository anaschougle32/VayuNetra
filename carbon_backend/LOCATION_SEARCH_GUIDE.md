## ‚úÖ How to Search and Select Locations

### **Method 1: Using the Search Box**

1. **Select "Other" from dropdown:**
   - Click on "Start Location" dropdown
   - Select "Other (Select on Map)"
   - OR click on "End Location" dropdown
   - Select "Other (Select on Map)"

2. **Search for location:**
   - The search box will automatically focus
   - Type the location name (e.g., "Thane Station", "Mumbai Airport")
   - Select from the autocomplete suggestions
   - OR press Enter after typing

3. **Location is set:**
   - Marker appears on map
   - Map centers on the location
   - Location is saved to the form

### **Method 2: Click on Map**

1. Select "Other" from dropdown
2. Click anywhere on the map
3. Marker moves to that location
4. You can drag the marker to adjust

### **Method 3: Use Saved Locations**

1. Select from dropdown:
   - "My Home" (your saved home location)
   - Any office location (from your employer)
2. Marker automatically appears at that location
